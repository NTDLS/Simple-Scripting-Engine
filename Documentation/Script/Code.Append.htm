Code :: Append


Description:
Reads a script into the interpreter engine. Allows for modularity and segmentation.
Syntax:
Code.Append(FileName as String)
Parameters:
  • FileName as String
    • File name to be appended to the end of the execution queue. This path is always relative to the path of the script that is including the new script. Any valid expression can be used.
Remarks:
Because the Code.Append function is executed sequentially as any other line of code, the included file will have its code appended to the end of the execution queue. The functions contained in the included file will be parsed and added to the function indexes but variables defined in the global scope will not be available to any previously added code.
Example:
<%...%>
<%

Code.Append("Functions.ss")

%>
