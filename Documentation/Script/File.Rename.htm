File :: Rename


Description:
Renames a file in the filesystem, can also be used to move a file.
Syntax:
File.Rename(OldFileName as String, NewFileName as String) as Numeric
Parameters:
  • OldFileName as String
    • Original file name and path.
  • NewFileName as String
    • Name and path to rename the file to.
Example:
<%...%>
<%

File.Rename("C:\\Source.txt", "C:\\target.txt")

%>
