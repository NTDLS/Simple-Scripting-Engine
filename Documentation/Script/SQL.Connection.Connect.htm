SQL :: Connection :: Connect


Description:
Connects to an instance of SQL Server.
Syntax:
SQL.Connection.Connect(ServerName as String, DefaultDatabase as String = "master", UserName as String = NULL, Password as String = NULL) as Numeric
Parameters:
  • ServerName as String
    • Name of the SQL Server to connect to
  • DefaultDatabase as String
    • Optional database name to connect to
  • UserName as String
    • Optional username to use for authentication, trusted authorization will be used if a username is not specified
  • Password as String
    • Optional password to use for authentication
Example:
<%...%>
<%

var sqlConn as SQL.Connection

sqlConn.Connect("(local)")

;...

sqlConn.Close();

%>
