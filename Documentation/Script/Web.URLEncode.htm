Web :: URLEncode


Description:
Converts the given input text to a URL encoded string, making it safe to pass/post from page to page via query string etc.
Syntax:
Web.URLEncode(Text as String, Length as Numeric = -1) as String
Parameters:
  • Text as String
    • The input text to be encoded
  • Length as Numeric
    • Optional length of the input text
Example:
<%...%>
<%

Print(Web.URLEncode("This Text Message is Directed @ You!"))

%>
