String :: IndexOf


Description:
Returns the index of the an occurrence of a string inside of another string.
Syntax:
String.IndexOf(StringToFind as String, CaseSensitive as Numeric = 1, StartPosition as Numeric = 0) as Numeric
Parameters:
  • StringToFind as String
    • The string to find
  • CaseSensitive as Numeric
    • Denotes whether the search will be case sensitive. The default is True.
  • StartPosition as Numeric
    • Zero bases character position where the search starts. If omitted, the search starts at the start of string1, position 0.
Example:
<%...%>
<%

Print("Hello Cruel World".IndexOf("Cruel"))

%>
