Socket :: Client :: Send


Description:
Adds data to the queue of data to be sent to the remote peer. This call is synchronous and will block until the data is successfully added to the send queue. The function will return FALSE if the data could not be sent due to a disconnection or other internal error.
Syntax:
Socket.Client.Send(Data as String) as Numeric
Parameters:
  • Data as String
    • Data to send to the remote peer
