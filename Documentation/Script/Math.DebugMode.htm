Math :: DebugMode


Description:
Allows for enabling or disabling debugging of the internal mathematics engine. When mathematical debugging is enabled, the engine will "show its work" in the output windows of the IDE. this function returns the previous setting.
Syntax:
Math.DebugMode(DebugMode as Numeric) as Numeric
Parameters:
  • DebugMode as Numeric
    • Boolean value which defines wheather to enable or disable mathematical debugging.
Example:
<%...%>
<%

Math.DebugMode(true)

If (10 * 10 = 25 * 4) ...
If(10 * 10 = 25 * 4)
    ;Do something...
End If
    
%>
