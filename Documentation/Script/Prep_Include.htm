#Include


Description:
Reads a script into the interpreter engine. Allows for modularity and segmentation.
Syntax:
#Include FileName
Parameters:
  • FileName
    • The name of the variable to define.
  • Value
    • File name to be included. This path is always relative to the path of the script that is including the new script. Any valid expression can be used.
Remarks:
Scripts that are included with this preprocessor are added inline and expanded in place of the actual #include tag. The code in the included script is executed as if it were in the same script file as it was included from.
Example:
<%...%>
<%

#Include "FileName.ss"

%>
