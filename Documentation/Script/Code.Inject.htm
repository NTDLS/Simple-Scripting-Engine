Code :: Inject


Description:
Injects code into the interpreter engine.
Syntax:
Code.Inject(CodeText as String) as Numeric
Parameters:
  • CodeText as String
    • Line of code to be injected into the interpreter engine.
Example:
<%...%>
<%

Code.Inject("Function InjectedTestProc(sText as String) as Numeric" _
    & CrLf & "Return(sText & \", and this was passed out\")" _
    & CrLf & "End Function")

Print(InjectedTestProc("This was passed in")) 

%>
