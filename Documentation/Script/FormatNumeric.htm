FormatNumeric


Description:
Converts a numeric value or string into a friendly formatted string.
Syntax:
FormatNumeric(Value as Numeric, DecimalPlaces as Numeric = 0) as String
Parameters:
  • Value as Numeric
    • Numeric value (or string that can be converted to a numeric) whose value is to be formatted.
  • DecimalPlaces as Numeric
    • Number of decimal places to use in the conversion.
Example:
<%...%>
<%

Print(FormatNumeric(123456.789, 2)) 

%>
