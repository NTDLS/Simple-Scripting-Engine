Error :: Line


Description:
Retrieves the script file code line that the soft error occurred in.
Syntax:
Error.Line(Index as Numeric = 0) as Numeric
Parameters:
  • Index as Numeric
    • The index of the soft error in the error stack.
Example:
<%...%>
<%
