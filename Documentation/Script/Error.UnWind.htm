Error :: UnWind


Description:
Causes any error (hard or soft) that are currently on the stack to be immediately thrown.
Syntax:
Error.UnWind() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    Sleep(-100) ;This will produce the error "Sleep time must be greater than zero".
    Error.UnWind() ;This call will actually throw the error from above. 

    %>
