XML :: Reader :: ProgressiveScan


Description:
Causes the XML reader to always find the next element in a position greater than the position of the previous found element. This functionality is only applicable to the ToReader() reader property method and is designed to allow for simple iteration through an XML document.
Syntax:
XML.Reader.ProgressiveScan(Enable as Numeric)
Parameters:
  • Enable as Numeric
    • Whether or not to enable progressive scanning on the XML reader.
