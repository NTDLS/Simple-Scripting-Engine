List :: DeleteAll


Description:
Searches the list for a given value and deletes every occurrence of it. Returns the number of occurrences which were found and removed.
Syntax:
List.DeleteAll(Value as String) as String
Parameters:
  • Value as String
    • The value to search for and remove.
Example:
<%...%>
<%

var listItems as List

Print("Adding Items.")
listItems.Add("Hello")
listItems.Add("Cruel")
listItems.Add("Cruel")
listItems.Add("Cruel")
listItems.Add("Cruel")
listItems.Add("Cruel")
listItems.Add("World")

Print("\nEnumeration before remove:")
For (i as Numeric to listItems.Count()) ...
For(i as Numeric to listItems.Count())
    Print("\tList[" & i & "]: " & listItems[i])
Next

listItems.DeleteAll("Cruel")

Print("\nEnumeration after remove:")
For (i as Numeric to listItems.Count()) ...
For(i as Numeric to listItems.Count())
    Print("\tList[" & i & "]: " & listItems[i])
Next

%>
