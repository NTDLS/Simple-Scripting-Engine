String :: Equals


Description:
Determines if two strings are equal.
Syntax:
String.Equals(String as String, CaseSensitive as Numeric = 1) as Numeric
Parameters:
  • String as String
    • String to be compared to the base.
  • CaseSensitive as Numeric
    • Determines if the comparison will be case sensitive.
