Prints


Description:
The prints function provides basic console output. Does not automatically print a trailing new line character.
Syntax:
Prints(...)
Parameters:
Infinite parameters of any type.
Remarks:
This function will take an infinite number of parameters. All parameters passed to this function will be printed to the console.

This function differs from the Print() function as it does not print a trailing line feed (new line) character.
Example:
<%...%>
<%

Prints("This is a Number: " & 10 & ", and this is text.") 

%>
