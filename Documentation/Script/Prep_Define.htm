#Define


Description:
Defines a global preprocessed variable.
Syntax:
#Define VariableName Value
Parameters:
  • VariableName
    • The name of the variable to define.
  • Value
    • The value that should be assigned to the variable once defined. Any valid expression can be used.
Example:
<%...%>
<%

#Define VARABLE_NAME 0xFFF

%>
