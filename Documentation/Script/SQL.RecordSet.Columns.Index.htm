SQL :: RecordSet :: Columns :: Index


Description:
Retrieves the index of a column in the record set that was returned from SQL Server.
Syntax:
SQL.RecordSet.Columns.Index(ColumnName as String) as Numeric
Parameters:
  • ColumnName as String
    • The name or alias of the column.
Example:
<%...%>
<%

var Cn2 as SQL.Connection
var RS as SQL.RecordSet

Cn2.Connect("DRIVER={ODBC Driver 11 for SQL Server};DATABASE=master;SERVER=(local);Trusted_Connection=yes;")

Cn2.Execute("SELECT 'A' as A, 'B' as B, 'C' as C", @RS)

print("Index of Name A:" & RS.Columns.Index("A"))

%>
