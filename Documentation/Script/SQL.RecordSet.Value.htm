SQL :: RecordSet :: Value


Description:
Retrieves a value from SQL Server through the bound record set object.
Syntax:
SQL.RecordSet.Value(ColumnName as String) as Numeric
Parameters:
  • ColumnName as String
    • Column name or numeric column index to retrieve data from
