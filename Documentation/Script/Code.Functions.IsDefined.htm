Code :: Functions :: IsDefined


Description:
Determines if a specified function exists.
Syntax:
Code.Functions.IsDefined(FunctionName as String) as Numeric
Parameters:
  • FunctionName as String
    • The name of the function to be tested for existence.
Example:
<%...%>
<%

Function OneThousand() as Numeric ...
Function OneThousand() as Numeric
    Return(1000)
End Function

If(Code.Functions.IsDefined("OneThousand"))
