Numeric :: ToString


Description:
Converts a numeric value to a string with optional precision.
Syntax:
Numeric.ToString(DecimalPlaces as Numeric = -1) as String
Parameters:
  • DecimalPlaces as Numeric
    • Optional number of decimal-places of precision. Default value is -1, which results in no change of precision.
Example:
<%...%>
<%

var iValue as Numeric = 3.14159

Print(iValue.ToString())
Print(iValue.ToString(2))

%>
