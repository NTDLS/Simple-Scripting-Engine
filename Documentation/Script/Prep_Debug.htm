#Debug


Description:
Programmatically enables the internal debugging engine allowing the debugger (IDE) to attach to the running script process and evaluate it's execution.
Syntax:
#Debug
Parameters:
Remarks: This option is given only as a pre-processor because debugging was not intended to be toggled on and off, nor was it intended to be enabled later than pre-execution of user code.
Example:
<%...%>
<%

#debug

While (true) ...
while(true)
    Print(Sys.TickCount())
    Sleep(100)
WEnd

%>
