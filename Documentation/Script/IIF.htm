IIF


Description:
Evaluates an expression and returns a value based on whether the given expression evaluates to true or false.
Syntax:
IIF(Logic as Numeric, ValueIfTrue as String, ValueIfFalse as String) as String
Parameters:
  • Logic as Numeric
    • The expression to be evaluated.
  • ValueIfTrue as String
    • The value to return if the expression evaluates to true.
  • ValueIfFalse as String
    • The value to return if the expression evaluates to false.
Example:
<%...%>
<%

Print(IIF(10 * 10 = 100, "Yes", "No"))
Print(IIF(10 * 10 = 50, "Yes", "No"))
Print(IIF(true = false, "Yes", "No"))
Print(IIF(true = true, "Yes", "No"))

%>
