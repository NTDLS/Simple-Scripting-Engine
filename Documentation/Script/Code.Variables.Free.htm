Code :: Variables :: Free


Description:
Frees all resources associated with a variable. The variable can then be redeclared as any type.
Syntax:
Code.Variables.Free()
Parameters:
    Example:
    <%...%>
    <%

    Var DataBuffer as String = "Hello World"
    Print("DataBuffer: " & DataBuffer)

    Code.Variables.Free(@DataBuffer)

    Var DataBuffer as Numeric = 10 * 10
    Print("DataBuffer: " & DataBuffer)

    %>
