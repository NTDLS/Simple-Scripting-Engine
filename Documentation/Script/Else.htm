Else


Description:
The ELSE statement is a supplement to the IF statement as it allows for the execution of a alternate block of code in the event that an IF expression evaluates to false.
Example:
<%...%>
<%

;--------------------------------------------------------------------------

If (5 > 10) ...
If(5 > 10)
    Print("The expression is true")
Else
    ;5 IS NOT greater than 5, so we will hit this line.
    Print("The expression is false")
End If

;--------------------------------------------------------------------------

If (10 > 5) ...
If(10 > 5)
    ;10 IS greater than 5, so we will hit this line.
    Print("The expression is true")
Else
    Print("The expression is false")
End If

;--------------------------------------------------------------------------

;Neither of the following are true, so we will not execute anything other than the expressions.
If (5 > 10) ...
If(5 > 10)
    Print("The expression is true")
Else If(10 > 20)
    Print("The expression is false")
End If

;--------------------------------------------------------------------------

;Neither of the following are true, but we have an ELSE so it will be executed.
If (5 > 10) ...
If(5 > 10)
    Print("The expression is true")
Else If(10 > 20)
    Print("The expression is false")
Else
    Print("This is the default statement")
End If

%>
