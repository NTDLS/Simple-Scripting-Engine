Checksum


Description:
Generates a standard CRC32 checksum for the given input text data.
Syntax:
Checksum(Text as String, Length as Numeric = -1) as Numeric
Parameters:
  • Text as String
    • Input text.
  • Length as Numeric
    • Input text length.
Remarks:
The checksum function generates a CRC32 compatible with that of PKZip, WinZip and Ethernet using the 0x04C11DB7 polynomial
Example:
<%...%>
<%

Print(Checksum("This is a test")) 

%>
