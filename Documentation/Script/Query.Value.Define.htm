Query :: Value :: Define


Description:
Allows for the dynamic definition of Query String Variables.
Syntax:
Query.Value.Define(FieldName as String, FieldValue as String = NULL) as Numeric
Parameters:
  • FieldName as String
    • Name of the variable to define
  • FieldValue as String
    • Optional value for the variable being defined.
Example:
<%...%>
<%

If (not Form.Value.IsDefined("Firstname")) ...
If(not Form.Value.IsDefined("Firstname"))
    Form.Value.Define("Firstname", "Jim")
End If

Print(Form.Value.ByName("Firstname"))

%>
