Settings :: ThrowSoftExceptions


Description:
Disables automatic throwing of soft (non-critical) errors. This can, in some cases, allow for more granular control of script execution.
Syntax:
Settings.ThrowSoftExceptions(Value as Numeric = -1) as Numeric
Parameters:
  • Value as Numeric
    • An optional true or false value. Enables or Disables throwing of soft errors.
Remarks:
Returns the previous value of Throw Soft Exceptions. Also, Errors.Clear() clears the soft exception stack - if it is not called, the error stack will continue to grow - it is therefore recommended that ThrowSoftExceptions only be disabled temporarily.
Example:
<%...%>
<%

Settings.ThrowSoftExceptions(false)
Sleep(-100) ;This will produce a "soft" error "Sleep time must be greater than zero"

If (Error.Count()) 'Did an error occur? ...
If(Error.Count()) ;Did an error occur?

    Print(Error.Count() & " error(s) occured!")

    ;Loop through all of the errors that occured
    For (iIndex as Numeric to Error.Count()) ...
    For(iIndex as Numeric to Error.Count())
        Print(Tab & iIndex + 1 & "): " & Error.Text(iIndex) & " on line " & Error.Line(iIndex))
    Next

    Error.Clear() ;Clear the errors.
Else
    Print("No errors occured.")
End If

Settings.ThrowSoftExceptions(true)

%>
