Socket :: Server :: Start


Description:
This function is used to start the client or server. This call is required before any incoming connections can be accepted and is also required before any outgoing connections can be made.
Syntax:
Socket.Server.Start(MaxConnections as Numeric = 10, ListenPort as Numeric = 0) as Numeric
Parameters:
  • MaxConnections as Numeric
    • The maximum number of connections that can be concurrently held at any given time.
    • ListenPort as Numeric
    • Optional listening port for the local server. If this parameter is not specified then the socket server will not bind to any local port and can only be used to make outgoing connections.
