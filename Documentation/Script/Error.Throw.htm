Error :: Throw


Description:
Adds a soft error to the error stack, this error can either be caught by a Try / Catch or by using Error.Count().
Syntax:
Error.Throw(Text as String = "") as Numeric
Parameters:
  • Text as String
    • Text describing the custom error.
Example:
<%...%>
<%

Try  ...
Try 
    Error.Throw("The user does not exist.")
Catch(ex)
    Print("Error: [" & ex.ErrorText() & "]" & CrLf & Tab _
        & "in [" & ex.ErrorFile() & "] on line " & ex.ErrorLine())
End Try

%>
