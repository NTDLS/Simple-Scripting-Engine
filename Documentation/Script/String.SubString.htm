String :: SubString


Description:
Returns a substring containing a specified number of characters from a string.
Syntax:
String.SubString(StartIndex as Numeric, Length as Numeric = NULL) as String
Parameters:
  • StartIndex as Numeric
    • The zero based starting index to derive the substring
  • Length as Numeric
    • The number of characters from the starting index to return. This parameter is optional, if not specified, the function will return the remainder of the text after the starting index
    • Example:
    <%...%>
    <%

    var Text as String = "Hello cruel, CRUEL World"
    Print(Text.SubString(6))
    Print(Text.SubString(6, 5))

    %>
