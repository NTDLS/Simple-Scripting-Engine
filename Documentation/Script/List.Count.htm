List :: Count


Description:
Returns the number of allocated list items which are in use.
Syntax:
List.Count() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    var listItems as List

    Print("Adding Items.")
    listItems.Add("Hello")
    listItems.Add("Cruel")
    listItems.Add("World")

    Print("Item Count: " & listItems.Count())
