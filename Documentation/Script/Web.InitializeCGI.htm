Web :: InitializeCGI


Description:
Initializes web form variables, loads environment variables and maps variables to these values.
Syntax:
Web.InitializeCGI()
Parameters:
    Remarks:
    This function is called automatically when one of the following environment variables is present: GATEWAY_INTERFACE, SERVER_PROTOCOL, SERVER_SOFTWARE, SCRIPT_NAME.
    Example:
    <%...%>
    <%

    Web.InitializeCGI() 

    %>
