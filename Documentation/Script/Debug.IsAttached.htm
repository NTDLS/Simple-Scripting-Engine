Debug :: IsAttached


Description:
Returns true if the debugger is attached to the script. This is true when the script was ran from within or later attached to the script process.
Syntax:
Debug.IsAttached() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    If (Debug.IsAttached()) ...
    If(Debug.IsAttached())
        Debug.Evaluate("Sys.TickCount()")
    Else
        Print(Sys.TickCount())
    End If

    %>
