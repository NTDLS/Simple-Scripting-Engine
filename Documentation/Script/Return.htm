Return


Description:
Returns data to a calling procedure / function.
Syntax:
Return(...) as String
Parameters:
Infinite parameters of any type.
Example:
<%...%>
<%

Function MultiplyByTen(iValue as Numeric) as Numeric ...
Function MultiplyByTen(iValue as Numeric) as Numeric
    Return(iValue * 10)
End Function

Print(MultiplyByTen(50)) 

%>
