#ContentType


Description:
Sets the content type for the scripts output. This preprocessor can only be defined one time and is written to the STDOUT before any other values (with the sole exception of parsing errors).
Syntax:
#ContentType ContentTypeText
Parameters:
  • ContentTypeText
    • Text denoting the output content type.
Example:
<%...%>
<%

#ContentType "text/html"

Print("Hello from a CGI process!")

%>
