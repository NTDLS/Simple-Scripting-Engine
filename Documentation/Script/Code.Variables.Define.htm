Code :: Variables :: Define


Description:
Defines / declares a variable.
Syntax:
Code.Variables.Define(VariableName as String, TypeName as String, Expression as String = NULL, Scope as Numeric = -1) as Numeric
Parameters:
  • VariableName as String
    • The name of the variable to be defined.
  • TypeName as String
    • name of the type to define the variable as.
  • Expression as String
    • Optional textual expression to be parsed to determine th einitial value of the variable.
  • Scope as Numeric
    • Optional variable scope. this is used to determine the accessible scope and the life-span of the variable. Negative values (the default) map to current scope.
Example:
<%...%>
<%
Code.Variables.Define("VarName", "String", "FormatNumeric((((0xFF*0xFF) % 100) * 200) * 0.2048, 2)")

Print("Value of [VarName]: " & VarName)
%>
