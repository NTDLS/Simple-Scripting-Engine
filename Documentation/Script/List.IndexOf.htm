List :: IndexOf


Description:
Searches the list and returns the index of the first found occurrence for the given value.
Syntax:
List.IndexOf(Value as String) as String
Parameters:
  • Value as String
    • The value to search the list for.
Example:
<%...%>
<%

var listItems as List

Print("Adding Items.")
listItems.Add("Hello")
listItems.Add("Cruel")
listItems.Add("World")

Print("Index of Cruel: " & listItems.IndexOf("Cruel") )
%>
