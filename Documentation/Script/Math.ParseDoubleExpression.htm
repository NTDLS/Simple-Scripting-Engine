Math :: ParseDoubleExpression


Description:
Parses a string representation of mathematical equation and returns the result in the form of an floating point value.
Syntax:
Math.ParseDoubleExpression(Expression as String) as Numeric
Parameters:
  • Expression as String
    • The string to be parsed.
Example:
<%...%>
<%

Print("ParseDoubleExpression: " & Math.Round(Math.ParseDoubleExpression("10 * 9.99"), 4)))

%>
