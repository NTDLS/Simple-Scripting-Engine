File :: Size


Description:
Returns the size of the specified file, in bytes.
Syntax:
File.Size(FileName as String = NULL) as Numeric
Parameters:
  • FileName as String
    • Optional path to a file.
Example:
<%...%>
<%

Var txtFile As File
If (txtFile.Open("C:\\Test.txt", FileAccess.Write)) ...
If(txtFile.Open("C:\\Test.txt", FileAccess.Write))

    Print("File size by handle: " & txtFile.Size())

    txtFile.Close()
End If

Print("File size by name: " & File.Size("C:\\Test.txt"))

File.Delete("C:\\Test.txt")
