Debug :: Evaluate


Description:
Allows you to evaluate an expression in real-time when running in the IDE.
Syntax:
Debug.Evaluate(CodeText as String) as Numeric
Parameters:
  • CodeText as String
    • Any valid expression. Functions, variables, numbers and strings.
Remarks:
1) This function is completely ignored when running outside of the debugger.
2) This function has no scope limitations and can therefore report the values of variables from calling functions.
Example:

For (i as Numeric to 10000) ... 
For(i as Numeric to 10000)
