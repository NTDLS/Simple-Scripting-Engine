Break


Description:
Breaks out of a [While] or [For] loop.
Syntax:
Break
Parameters:
Example:
<%...%>
<%

Print("Before For")

For (i as numeric = 0 to 5) ...
For(i as numeric = 0 to 5)
    Print("Before If: " & i)
    If (i = 2) ...
    If(i = 2)
        Print("Before Break: " & i)
        Break
        Print("After Break: " & i)
    End If
    Print("After If: " & i)
Next

Print("After For")

%>
