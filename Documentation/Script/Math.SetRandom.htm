Math :: SetRandom


Description:
Set the seed for the pseudo-random number generator.
Syntax:
Math.SetRandom(Seed as Numeric = NULL) as Numeric
Parameters:
  • Seed as Numeric
    • Optional initial seed value for the random number generator
Remarks:
If no seed is passed into this function, the TickCount() is used
Example:
<%...%>
<%

Math.SetRandom(64654) 

%>
