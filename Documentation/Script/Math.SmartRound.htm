Math :: SmartRound


Description:
Floating-Point arithmetic is imprecise, SmartRound was designed to round-up and/or truncate the noise created from this arithmetic.
Syntax:
Math.SmartRound(Expression as Numeric) as Numeric
Parameters:
  • Expression as Numeric
    • The floating point value to be rounded.
Example:
<%...%>
<%

Print("Smart Round: " & Math.SmartRound(99.123460000000010))

%>
