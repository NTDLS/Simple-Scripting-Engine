SQL :: RecordSet :: Columns :: Name


Description:
Retrieves the name of a column in the record set that was returned from SQL Server.
Syntax:
SQL.RecordSet.Columns.Name(ColumnIndex as Numeric) as String
Parameters:
  • ColumnIndex as Numeric
    • The zero-bases index of the column.
Example:
<%...%>
<%

var Cn2 as SQL.Connection
var RS as SQL.RecordSet

Cn2.Connect("(local)")

Cn2.Execute("SELECT 'A' as A, 'B' as B, 'C' as C", @RS)

print("Name of Index 0:" & RS.Columns.Name(0))

%>
