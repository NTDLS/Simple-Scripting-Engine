Code :: Variables :: Undefine


Description:
Frees all resources associated with a variable. The variable can then be redeclared as any type.
Syntax:
Code.Variables.Undefine(VariableName as String) as Numeric
Parameters:
  • VariableName as String
    • Name of variable to be freed.
Example:
<%...%>
<%
var OneThousand as Numeric = 1000

Code.Variables.Undefine("OneThousand")

var OneThousand as String = "Hello World"

Print("Value of [OneThousand]: " & OneThousand)

%>
