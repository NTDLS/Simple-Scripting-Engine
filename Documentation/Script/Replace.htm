Replace


Description:
The Replace function replaces a specified part of a string with another string.
Syntax:
Replace(OriginalString as String, StringToReplace as String, With as String, CaseSensitive as Numeric = 1) as String
Parameters:
  • OriginalString as String
    • The string to be searched
  • StringToReplace as String
    • The part of the string that will be replaced
  • With as String
    • The replacement substring
  • CaseSensitive as Numeric
    • Denotes whether the search will be case sensitive. The default is True.
Example:
<%...%>
<%

var Text as String = "Hello cruel, CRUEL World"
Print(Replace(Text, "cruel", "lovely"))

%>
