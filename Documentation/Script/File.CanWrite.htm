File :: CanWrite


Description:
Determines is user under which the script is being run has write access to the specified file.
Syntax:
File.CanWrite(FileName as String) as Numeric
Parameters:
  • FileName as String
    • Path and file name of the file in which access is to be determined.
Example:
<%...%>
<%

If (File.CanWrite("C:\\FileToCheck.txt")) ...
If(File.CanWrite("C:\\FileToCheck.txt"))
    Print("The file CAN be accessed.")
Else
    Print("The file can NOT be accessed.")
End If

%>
