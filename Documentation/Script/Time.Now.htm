Time :: Now


Description:
Returns the current time in the optionally specified format.
Syntax:
Time.Now(FormatText as String = NULL) as String
Parameters:
  • FormatText as String
    • Optional time format
Remarks:
If no formats are specified, the function will returns the default format based on the computers locale settings.

Value Description
h Hours with no leading zero for single-digit hours; 12-hour clock
hh Hours with leading zero for single-digit hours; 12-hour clock
H Hours with no leading zero for single-digit hours; 24-hour clock
HH Hours with leading zero for single-digit hours; 24-hour clock
m Minutes with no leading zero for single-digit minutes
mm Minutes with leading zero for single-digit minutes
s Seconds with no leading zero for single-digit seconds
ss Seconds with leading zero for single-digit seconds
t One character time marker string, such as A or P
tt Multicharacter time marker string, such as AM or PM
Example:
<%...%>
<%

Print(Time.Now("hh|mm|ss")) 

%>
