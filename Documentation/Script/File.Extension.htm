File :: Extension


Description:
Returns the file extension of a given full path. This is the extension without the file name.
Syntax:
File.Extension(FullPath as String) as Numeric
Parameters:
  • FullPath as String
    • A full or partial file path. Physical existence is not required.
Example:
<%...%>
<%

Print(File.Extension("C:\\Program Files\\NetworkDLS\\Simple Scripting Engine\\SSE.exe")) 

%>
