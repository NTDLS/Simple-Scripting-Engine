Code :: Scope


Description:
Gets the current scope level, the scope is incremented for every function fall, loop and logical block and decremented when the end of these blocks are met or exited explicitly.
Syntax:
Code.Scope() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    Print("Top scope " & Code.Scope())
    For (iLoop as Numeric to 5) ...
    For(iLoop as Numeric to 5)
        Print("Loop scope " & Code.Scope())
        iLoop++
    Next

    Print("Bottom scope " & Code.Scope())

    %>
