SQL :: RecordSet :: Columns :: Count


Description:
Retrieves the count of columns that are represented in the record set that was returned from SQL Server.
Syntax:
SQL.RecordSet.Columns.Count() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    var Cn2 as SQL.Connection
    var RS as SQL.RecordSet

    Cn2.Connect("(local)")

    Cn2.Execute("SELECT 'A' as A, 'B' as B, 'C' as C", @RS)

    print("Column Count:" & RS.Columns.Count())


    %>
