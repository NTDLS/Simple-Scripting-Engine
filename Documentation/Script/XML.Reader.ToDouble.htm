XML :: Reader :: ToDouble


Description:
Returns an XML value as a double number value.
Syntax:
XML.Reader.ToDouble(TagName as String, DefaultValue as Numeric = 0) as Numeric
Parameters:
  • TagName as String
    • The name of the element to retrieve.
  • DefaultValue as Numeric
    • [optional] The default value to return if the tag is not found.
Example:
<%...%>
<%

var xmlText as String = "<People><Person><Id>101</Id><FirstName>Steve_101</FirstName><LastName>Jobs_101</LastName></Person></People>"
var people as XML.Reader

people.Parse(xmlText)

Print(people.ToDouble("Id"))

%>
