List :: Clear


Description:
Removes all items from the list. While this will deallocate the memory associated with storing each list item, it will not deallocate the slots which were allocated to store the items. (Roughly 4 bytes per list item).
Syntax:
List.Clear()
Parameters:
    Example:
    <%...%>
    <%

    var listItems as List

    Print("Adding Items.")
    listItems.Add("Hello")
    listItems.Add("Cruel")
    listItems.Add("World")

    Print("Count before clear: " & listItems.Count())

    listItems.Clear()

    Print("Count after clear: " & listItems.Count())

    %>
