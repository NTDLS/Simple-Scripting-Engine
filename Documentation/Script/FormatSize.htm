FormatSize


Description:
Formats a number into the standard computer shorthand (1 KB, 5.02 MB, ect...).
Syntax:
FormatSize(Value as Numeric, DecimalPlaces as Numeric = 0) as String
Parameters:
  • Value as Numeric
    • Input number
  • DecimalPlaces as Numeric
    • Number of decimal places for precision
Example:
<%...%>
<%

Print(FormatSize(1000))
Print(FormatSize(1024))
Print(FormatSize(1024*1024*512.5, 2))
Print(FormatSize(1024*1024*1024))
Print(FormatSize(1024.0*1024.0*1024.0*1024.0*7.5, 2))

%>
