Code :: Variables :: IsDefined


Description:
Determines if a specified variable exists and is accessible in the current scope.
Syntax:
Code.Variables.IsDefined(VariableName as String) as Numeric
Parameters:
  • VariableName as String
    • The name of the variable to be tested for existence.
Example:
<%...%>
<%
var OneThousand as Numeric = 1000

If (Code.Variables.IsDefined("OneThousand")) ...
If(Code.Variables.IsDefined("OneThousand"))
    Print("Value of [OneThousand]: " & OneThousand)
Else
    Print("The variable is not defined.")
End If

%>
