Exception :: SetText


Description:
Sets the text of an exception object for custom error handling.
Syntax:
Exception.SetText(Text as String)
Parameters:
  • Text as String
    • The text message of the exception.
Example:
<%...%>
<%

var ex as Exception

ex.SetText("A sample exception")

Print("[" & ex.GetText() & "] occured on line " _
