Code :: Functions :: Call


Description:
This function allows for the dynamic execution of script function expressions.
Syntax:
Code.Functions.Call(DynamicExpression as String) as String
Parameters:
  • DynamicExpression as String
    • The expression that evaluates to a a function and its parameters.
Example:
<%...%>
<%

var functionName as String = "TestFunction"
var returnValue as String 

returnValue = Code.Functions.Call("Dynamic" & functionName & "(\"This is the functions input\")")

Print("The return value was: " & returnValue)
