List :: Contains


Description:
Determines if given a value exists in the list.
Syntax:
List.Contains(Value as String) as String
Parameters:
  • Value as String
    • The value of any type to search for in the list.
Example:
<%...%>
<%

var listItems as List

Print("Adding Items.")
listItems.Add("Hello")
listItems.Add("Cruel")
listItems.Add("World")

Print("Contains 'Cruel': " & IIF(listItems.Contains("Cruel"), "Yes", "No"))
Print("Contains 'Plesant': " & IIF(listItems.Contains("Plesant"), "Yes", "No"))

%>
