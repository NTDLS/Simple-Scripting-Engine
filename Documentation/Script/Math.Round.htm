Math :: Round


Description:
Founds a floating point value to the specified number of decimal places.
Syntax:
Math.Round(Expression as Numeric, Decimals as Numeric = 0) as Numeric
Parameters:
  • Expression as Numeric
    • The value to round.
  • Decimals as Numeric
    • The number of decimal places to round the value to.
Example:
<%...%>
<%

Print("Round: " & Math.Round(99.1259123, 4))
Print("Round: " & Math.Round(99.1259123, 3))

%>
