Env :: Count


Description:
Retrieves the count of environment variables in the internal environment variable array. This count is useful for iterating through the environment variables.
Syntax:
Env.Count() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    For (i as Numeric = 0 to Env.Count()) ...
    For(i as Numeric = 0 to Env.Count())
        Print(Env.Name(i) & "=" Env.Value(i))
    Next

    %>
