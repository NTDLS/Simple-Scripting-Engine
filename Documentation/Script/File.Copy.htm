File :: Copy


Description:
Copies a file at one location in the filesystem to another location.
Syntax:
File.Copy(OldFileName as String, NewFileName as String, FailIfExists as Numeric = 1) as Numeric
Parameters:
  • OldFileName as String
    • Source file name and or path where file will be copied from.
  • NewFileName as String
    • Target file name and or path where file will be copied to.
  • FailIfExists as Numeric
    • TRUE to fail if the target alreasy exists. FALSE will cause the destination to be overwritten.
Example:
<%...%>
<%

File.Copy("C:\\Source.txt", "C:\\target.txt")

%>
