List :: Allocated


Description:
For performance purposes, list slots are allocated ahead of time. As the list fills up, new slots are allocated at 10% of the size of the total list. This function will return the number of allocated (but not necessarily used) list slots.
Syntax:
List.Allocated() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    var listItems as List

    Print("Adding Items.")
    listItems.Add("Hello")
    listItems.Add("Cruel")
    listItems.Add("World")

    Print("Allocated: " & listItems.Allocated())

    %>
