Trim


Description:
Trims or removes leading and trailing whitespace from a string.
Syntax:
Trim(Text as String, Length as Numeric = -1) as Numeric
Parameters:
  • Text as String
    • Input string containing extra whitespace.
  • Length as Numeric
    • Optional length of the input string. If not specified, the value will be calculated at runtime.
Example:
<%...%>
<%

Print("[" & Trim("    Hello World    ") & "]")
Print("[" & Trim("    Hello World    ", 5) & "]")

%>
