File :: Handle


Description:
Returns the numeric handle value of a previously opened file.
Syntax:
File.Handle() as Numeric
Parameters:
    Example:
    <%...%>
    <%

    Var txtFile As File

    If (txtFile.Open("C:\\Test.txt", FileAccess.Write)) ...
    If(txtFile.Open("C:\\Test.txt", FileAccess.Write))

        Print("Handle: " & txtFile.Handle())

        txtFile.Close()
    End If

    File.Delete("C:\\Test.txt")

    %>
