List :: DeleteAt


Description:
Removes an item from the list at the given index/position. Returns true if the item was found and deleted, otherwise returns false.
Syntax:
List.DeleteAt(Index as Numeric) as String
Parameters:
  • Index as Numeric
    • The index of the value to remove.
Example:
<%...%>
<%

var listItems as List

Print("Adding Items.")
listItems.Add("Hello")
listItems.Add("Cruel")
listItems.Add("World")

Print("\nEnumeration before remove:")
For (i as Numeric to listItems.Count()) ...
For(i as Numeric to listItems.Count())
    Print("\tList[" & i & "]: " & listItems[i])
Next

listItems.DeleteAt(1)

Print("\nEnumeration after remove:")
For (i as Numeric to listItems.Count()) ...
For(i as Numeric to listItems.Count())
    Print("\tList[" & i & "]: " & listItems[i])
Next

%>
